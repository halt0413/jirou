# 58ハッカソン

## タイトル　
二郎らーめん専用アプリ

## 概要
mapで二郎らーめんの店舗を表示、二郎らーめん特有のコールをアプリで出力、プロフィール機能

## 開発背景
初めて行く人・初心者には二郎らーめんに行くにあたってわからないこと（用語やコール）が多いと感じ、
このわからないことを初めてでも解消できるものがあるといいなと思い開発。
